# Black Hole Dynamo Radius Calculator

This Python script calculates the **inner dynamo radius** for a range of black hole spins and plots the results.  
It models a magnetohydrodynamic (MHD) dynamo in the vicinity of a Kerr black hole and determines the radius where the dynamo becomes active.  

## Features

- Computes the maximum growth rate of the dynamo (`lambda_max`) for different black hole spins.
- Determines the first positive eigenvalue radius corresponding to the onset of dynamo activity.
- Plots the **inner dynamo radius** as a function of spin alongside the black hole's event horizon.
